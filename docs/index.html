Tools

Seven tools. One toolbar.

Each tool is a focused instrument — no menus to dig through, no settings panels in the way.

IInspect

Hover to highlight any element, click to pin it. Shows tag, class, font, and color. One-click CSS copy.

TTypography

Hover any element to see its font family, size, and weight — live, without opening DevTools.

EEdit Text

Double-click any text to edit it inline. Enter to save, Esc to cancel. Great for copy reviews.

/Search

Find elements by CSS selector or text content. Enter and Shift+Enter cycle through matches.

RRulers

Drop full-screen guide lines. Select one line, hover another — the gap displays automatically.

OOutlines

Toggle a 1px outline on every element on the page to see the box structure at a glance.

ZZoom

Scale the page in or out in 10% steps. Resets cleanly when you switch to another tool.

Get started

Load it in under a minute

No Chrome Web Store. Load directly from the repo as an unpacked extension — takes three steps.

01

Download the ZIP and unzip it, or clone the repo with git clone https://github.com/maskmanlucifer/argus.git

02

Open chrome://extensions, enable Developer mode in the top-right corner, then click Load unpacked and select the argus/ folder.

03

Pin the extension from the toolbar menu and click its icon on any tab. The inspector opens immediately.
